# useImmer

一个使用 [immer](https://github.com/mweststrate/immer) 操作状态的 hook。

## 安装

```bash

npm i @vue-hooks-plus/use-immer

```

> 该 `hook` 基于 `immer` 管理状态，会安装 `immer`保证在项目中正常工作

> 独立安装 `@vue-hooks-plus/use-immer`
