# 依赖刷新

`useRequest` 提供了一个 `options.refreshDeps` 参数，当它的值变化后，会重新触发请求。

## 代码演示

<demo src="./demo/demo.vue"

## API

### Options

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| refreshDeps | 依赖响应式对象,和 `vue` 的 `watch` 传入监听的对象用法一致 | `any` \| `WatchSource[]` | `-` |
